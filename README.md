# 🐣 Agent Pet - Telegram Web App

> Tamagotchi-style coding companion for Telegram

![Demo](demo.png)

## 🎮 Features

- 🥚 5 Evolution Stages (Egg → Baby → Teen → Adult → Legend)
- 🍕 Feed, 🎮 Play, 💻 Code mechanics
- 📊 Real-time stats (Hunger, Happiness, Energy)
- 🎨 Pixel art design
- 📱 Telegram Mini App integration
- ⚡ Haptic feedback

## 🚀 Setup

### 1. Create Bot
```bash
# Talk to @BotFather
/newbot
# Get your BOT_TOKEN
```

### 2. Configure
```bash
cp .env.example .env
# Add your BOT_TOKEN and WEBAPP_URL
```

### 3. Deploy
```bash
npm install
npm start
```

## 🎯 Usage

1. Start bot: `/start`
2. Click "🎮 Open Pet" button
3. Take care of your pet!

## 📱 Telegram Mini App

- Works on all platforms (iOS, Android, Desktop)
- Native feel with Telegram UI
- Secure Web App environment
